
# Passenger Luggage Tax Calculator (Java)

This is a simple Java console application that calculates extra charges for passengers based on their luggage weight and category.

## Features
- Enter passenger category (First, Business, Economy)
- Enter luggage weight
- Calculates extra charge if luggage exceeds the allowed limit
- Keeps track of total extra charges
- Input validation (positive luggage weight, valid category)

## Technologies Used
- Java
- Loops
- Switch & If-Else statements
- Scanner

## How to Run
1. Open the project in a Java IDE (NetBeans, IntelliJ, or Eclipse)
2. Run the `main` method
3. Follow the instructions in the console
